# 📊 MLAnalysis

A machine learning project that applies both supervised and unsupervised models on two real-world datasets — Breast Cancer and Adults — and compares their performance.

---

## What it covers

- **Supervised:** SVM with Linear and RBF kernels, hyperparameter tuning
- **Unsupervised:** K-Means and Hierarchical Clustering (single, complete, average linkage)
- **Evaluation:** Accuracy, Precision, Recall, F1, ROC-AUC, Silhouette Score
- **Visualization:** Confusion matrices, ROC curves, dendrograms, PCA plots

---

## Datasets

| Dataset | Source |
|---------|--------|
| Breast Cancer Wisconsin | https://archive.ics.uci.edu/dataset/17/breast+cancer+wisconsin+diagnostic |
| Adults (Census Income) | https://archive.ics.uci.edu/dataset/2/adult |

Download both and place them in the same folder as the notebook before running.

---

## Built with

Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, SciPy

---

## How to run

1. Clone the repo
2. Download the datasets from the links above
3. Run `pip install pandas numpy scikit-learn matplotlib seaborn scipy`
4. Open `ML_Assignment_2.ipynb` and run all cells
